Overflow Behavior Testing

overflow: visible;

This is a long block of text meant to demonstrate how overflow behaves when content exceeds the size of the box. In visible mode, the text will continue outside the box without being clipped.

overflow: hidden;

This text is longer than the box height, so any extra content will be hidden. You won't see scrollbars, and users cannot access the hidden text. Only the visible portion fits inside.

overflow: scroll;

In scroll mode, scrollbars always appear even if the content fits. This ensures the user can scroll through the content but may look unnecessary if the text is short.
